% ---- 3.1 Asset Universe ----
% 15 risk ETFs + BIL (cash proxy)
% SPY, QQQ, VGK, EWJ, SCZ, EEM, VNQ, REM, GLD, IEF, TLT, TIP, DBC, BWX, RWX
% Justify breadth: 6 equity regions + real estate + commodities + bonds
% Source: Refinitiv Eikon, adjusted close prices
% Period: Jan 2008 – Dec 2025 | Train: 2008–2019 | OOS: 2020–2025

% ---- 3.2 Momentum Signal: 13612U ----
% M_i(t) = (1/4)(R_{1m} + R_{3m} + R_{6m} + R_{12m})
% Equal-weight average of 4 lookback windows
% Justification: reduces look-back sensitivity (Moskowitz et al., 2012)
% Absolute threshold θ = 2%: only eligible if M_i(t) ≥ 2%
% (Antonacci absolute momentum filter)

% ---- 3.3 Portfolio Selection: Minimum-Correlation Triplet ----
% Top-7 by momentum score → eligible set A_t+
% If |A_t+| ≥ 3: select triplet minimising avg pairwise correlation
%   Correlation computed on daily returns of previous calendar month
%   Equal-weight 1/3 allocation
% Motivation: Markowitz (1952) — diversification via low-correlation assets

% ---- 3.4 Defensive Mode ----
% Triggered when |A_t+| < 3 (not enough eligible assets)
% Enhanced defensive: TLT (50%) + IEF (50%) if avg momentum of TLT+IEF > 0
% Pure cash: BIL (100%) if momentum of defensive assets ≤ 0
% Rationale: flight-to-quality (long-duration treasuries outperform in risk-off)

% ---- 3.5 Market-Breadth Crash Filter ----
% Each month: compute r̄(t) = equal-weight avg return of all 15 risky ETFs
% If r̄(t) < τ = -4%: market crash signal → override to defensive mode
%   TLT+IEF if their momentum > 0, else BIL
% Threshold τ = -4% selected by in-sample optimisation (see Table X)
% Inspired by market breadth concept in Keller & Keuning (2017)

% ---- 3.6 Rebalancing and Execution ----
% Monthly frequency, end-of-month signal, executed first business day of next month
